# 🖼️ Editing image
Консольное приложение для обработки изображений с помощью команд на естественном языке. Используется локальная LLM через [Ollama](https://ollama.com/) для распознавания команды, затем OpenCV применяет соответствующий фильтр.

---

## Как это работает

```
Пользователь вводит текст → LLM (Ollama/qwen3) определяет команду → OpenCV обрабатывает изображение
```

Например, можно написать «сделай размытым» или «blur it» — модель сама определит нужную операцию.

---

## Доступные команды

| Команда | Действие |
|---|---|
| `rotate` | Поворот на 90° по часовой стрелке |
| `rotate_left` | Поворот на 90° против часовой стрелки |
| `gray` | Перевод в оттенки серого |
| `blur` | Размытие по Гауссу |
| `flip` | Зеркальное отражение по горизонтали |
| `brightness` | Увеличение яркости |
| `contrast` | Увеличение контрастности |
| `invert` | Инверсия цветов |
| `edges` | Выделение границ (Canny) |
| `sketch` | Эффект карандашного рисунка |

## Требования

- Python 3.8+
- [Ollama](https://ollama.com/) с загруженной моделью `qwen3`
- OpenCV

---

## Установка

```bash
# Клонировать репозиторий
git clone https://github.com/your-username/nl-image-processor.git
cd nl-image-processor

# Установить зависимости
pip install opencv-python requests

# Убедиться, что Ollama запущен и модель загружена
ollama pull qwen3
ollama serve
```

---

## Запуск

```bash
python main.py
```

Пример сессии:

```
Путь к изображению: photo.jpg
Введите команду: сделай чёрно-белым
Нейросеть определила команду: gray
Результат сохранён в result.jpg
```
